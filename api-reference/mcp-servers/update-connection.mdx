---
title: "Update MCP Server Connection"
openapi: "PUT /api/v1/mcp-servers/{server_id}"
---

Update an existing MCP server connection.

**Parameters:**

- `server_id` (path, required): ID of the MCP server connection to update

**Request Body:**

```json
{
  "api_key": "updated-api-key",
  "agent_enabled": false,
  "access_type": "read",
  "disabled_tools": [1, 2, 3]
}
```

**Response:**

```json
{
  "success": true,
  "data": {
    "id": 172,
    "mcp_server_id": 1,
    "mcp_server_name": "Pipedrive MCP Server",
    "api_key_verification_status": "verified",
    "mcp_server_access_type_name": "Read Access",
    "mcp_server_access_status_name": "Active",
    "agent_enabled": false,
    "disabled_tools": [1, 2, 3],
    "created_at": "2024-01-15T10:30:00Z"
  },
  "message": "MCP server connection updated successfully"
}
```
