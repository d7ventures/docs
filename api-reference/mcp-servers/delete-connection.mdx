---
title: "Delete MCP Server Connection"
openapi: "DELETE /api/v1/mcp-servers/{server_id}"
---

Delete an MCP server connection.

**Parameters:**

- `server_id` (path, required): ID of the MCP server connection to delete

**Response:**

```json
{
  "success": true,
  "data": null,
  "message": "MCP server connection deleted successfully"
}
```
