---
title: "Create MCP Server Connection"
openapi: "POST /api/v1/mcp-servers"
---

Create a new connection to an MCP server integration. Supports both API key and OAuth authentication methods.

**Request Body (API Key):**

```json
{
  "mcp_server": "pipedrive",
  "access_type": "write",
  "auth_method": "api_key",
  "api_key": "your-pipedrive-api-key",
  "agent_enabled": true
}
```

**Request Body (OAuth):**

```json
{
  "mcp_server": "pipedrive",
  "access_type": "write",
  "auth_method": "oauth",
  "agent_enabled": true
}
```

**Response (API Key):**

```json
{
  "success": true,
  "data": {
    "id": 273,
    "mcp_server_id": 1,
    "mcp_server_name": "Pipedrive MCP Server",
    "api_key_verification_status": "verified",
    "auth_method": "api_key",
    "mcp_server_access_type_name": "Write Access",
    "mcp_server_access_status_name": "Active",
    "agent_enabled": true,
    "disabled_tools": [],
    "created_at": "2024-01-20T15:30:00Z"
  },
  "message": "MCP server connection created successfully"
}
```
