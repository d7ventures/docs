---
title: "Get MCP Server Tools"
openapi: "GET /api/v1/mcp-servers/{server_id}/tools"
---

Retrieve all available tools for a specific MCP server connection. Returns tools in MCP (Model Context Protocol) format.

**Parameters:**

- `server_id` (path, required): ID of the MCP server connection
- `limit` (query, optional): Maximum number of tools to return (default: 50)
- `offset` (query, optional): Number of tools to skip for pagination (default: 0)
