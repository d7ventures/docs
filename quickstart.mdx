# Quickstart Guide

Welcome to Kambrium! This guide will help you get started with our MCP (Model Context Protocol) platform and connect your first AI application to SaaS tools.

## Prerequisites

Before you begin, make sure you have:

- A Kambrium account (sign up at [app.kambrium.com](https://app.kambrium.com))
- API credentials for the SaaS tools you want to connect
- Basic understanding of REST APIs and OAuth 2.1

## Step 1: Authentication Setup

First, set up authentication to access Kambrium's APIs:

<AccordionGroup>
  <Accordion icon="key" title="Get Your API Token">
    1. Log in to your [Kambrium Dashboard](https://app.kambrium.com) 2. Navigate
    to **Settings** ‚Üí **API Tokens** 3. Click **Generate New Token** 4. Copy and
    securely store your Personal Access Token (PAT)
    <Note>
      Keep your API token secure and never share it publicly. You'll use this
      token to authenticate all API requests.
    </Note>
  </Accordion>

  <Accordion icon="shield-check" title="Configure OAuth 2.1">
    For production applications, set up OAuth 2.1 authentication: 1. In your
    dashboard, go to **OAuth Applications** 2. Click **Create New Application**
    3. Configure your redirect URIs and scopes 4. Note your `client_id` and
    `client_secret` See our [OAuth 2.1 guide](/authentication/management-api)
    for detailed setup instructions.
  </Accordion>
</AccordionGroup>

## Step 2: Create Your First MCP Server

Create an MCP server instance to connect to your SaaS tools:

<AccordionGroup>
  <Accordion icon="server" title="Create MCP Server Instance">
    Use the Management API to create your first MCP server:
    
    ```bash
    curl -X POST https://api.kambrium.com/api/v1/mcp-servers \
      -H "Authorization: Bearer YOUR_API_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "server_type": "pipedrive",
        "configuration": {
          "api_token": "YOUR_PIPEDRIVE_TOKEN"
        }
      }'
    ```
    
    This creates a Pipedrive MCP server that your AI applications can connect to.
  </Accordion>
  
  <Accordion icon="list" title="List Available Tools">
    Once your server is created, list the available tools:
    
    ```bash
    curl -X GET https://api.kambrium.com/api/v1/mcp-servers/pipedrive/tools \
      -H "Authorization: Bearer YOUR_API_TOKEN"
    ```
    
    You'll see tools like `get_deals`, `create_contact`, `search_organizations`, etc.
  </Accordion>
</AccordionGroup>

## Step 3: Connect Your AI Application

Now connect your AI application to use the MCP server:

<AccordionGroup>
  <Accordion icon="plug" title="Initialize MCP Session">
    Start an MCP protocol session:
    
    ```python
    import requests
    
    # Initialize MCP session
    response = requests.post(
        "https://mcp.kambrium.com/initialize",
        headers={
            "Authorization": "Bearer YOUR_API_TOKEN",
            "Content-Type": "application/json"
        },
        json={
            "protocolVersion": "2025-03-26",
            "capabilities": {
                "tools": {},
                "resources": {}
            }
        }
    )
    
    session_id = response.json()["sessionId"]
    ```
  </Accordion>
  
  <Accordion icon="wrench" title="Call MCP Tools">
    Use the MCP protocol to call tools from your AI application:
    
    ```python
    # List available tools
    tools_response = requests.post(
        "https://mcp.kambrium.com/tools/list",
        headers={
            "Authorization": "Bearer YOUR_API_TOKEN",
            "X-Session-ID": session_id
        }
    )
    
    # Call a specific tool
    result = requests.post(
        "https://mcp.kambrium.com/tools/call",
        headers={
            "Authorization": "Bearer YOUR_API_TOKEN",
            "X-Session-ID": session_id
        },
        json={
            "name": "get_deals",
            "arguments": {
                "status": "open",
                "limit": 10
            }
        }
    )
    
    deals_data = result.json()
    ```
  </Accordion>
</AccordionGroup>
